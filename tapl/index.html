About

Goal of this library is to provide an easy and quick way of implementing perception pipelines. It provides methods for several computer vision algorithms such as visual odometry, graph optimization, feature detection, panaromic image stitching etc. TAPL also provides methods for point-based processing such as KD tree implementation, Euclidian Clustering.
